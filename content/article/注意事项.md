# Typst 博客写作注意事项

本文档记录了在 Typst 博客写作中常见的符号问题，以避免编译和部署失败。

## 1. 数学符号问题

### 1.1 内置函数作为单位使用

Typst 中某些符号名是内置函数，在数学模式中作为单位使用时必须用引号包裹：

| 错误写法 | 正确写法 | 说明 |
|---------|---------|------|
| `$rad^2$` | `$"rad"^2$` | `rad` 是弧度转换函数 |
| `$rad$` | `$"rad"$` | 同上 |

### 1.2 多字母下标

下标中包含多个字母或混合数字和字符串时，必须用引号包裹：

| 错误写法 | 正确写法 | 说明 |
|---------|---------|------|
| `$F_(vv)$` | `$F_("vv")$` | 多字母下标 |
| `$P_(xy)$` | `$P_("xy")$` | 同上 |
| `$z_(1,"true")$` | `$z_("1,true")$` | 混合数字和字符串 |
| `$P_("pred")$` | `$P_("pred")$` | 文本下标 |

### 1.3 特殊数学符号

#### 正负号 ±
```typst
$plus.minus 0.1$  # 正确
```

#### 圆圈乘号 ⊗ (张量积/四元数乘法)
```typst
$q_k times.circle e^(omega_k Delta t\/2)$  # 正确
```

#### 箭头符号
```typst
$Q$大 $arrow.r$ $P$大  # 正确，箭头前后需要空格
```

### 1.4 矩阵中的省略号

矩阵中的省略号必须明确指定方向：

| 错误写法 | 正确写法 | 说明 |
|---------|---------|------|
| `$mat(..., dots; ...)$` | `$mat(..., dots.h; ...)$` | 水平省略号用 `dots.h` |
| `$mat(...; dots.v, ...)$` | `$mat(...; dots.v, ...)$` | 垂直省略号用 `dots.v` |

完整示例：
```typst
$ P_0 = mat(P_r, 0, 0, dots.h; 0, P_q, 0, dots.h; 0, 0, P_v, dots.h; dots.v, dots.v, dots.v, P_(y_N)) $
```

## 2. 表格问题

### 2.1 表格中避免使用数学符号

表格单元格中直接使用数学符号可能导致编译错误：

**错误示例：**
```typst
#table(
  columns: 4,
  [*变量*], [*帧0值*], [*帧1预测值*], [*帧1更新值*],
  [协方差迹], [3.48], [approx 3.58], [approx 3.57],  # 错误：approx 是数学符号
)
```

**正确做法：** 使用列表或纯文本替代
```typst
*帧1预测值*：
- 协方差迹：约3.58
```

## 3. 常见错误对照表

| 错误信息 | 原因 | 解决方案 |
|---------|------|---------|
| `unknown variable: rad` | `rad` 是内置函数 | 改为 `$"rad"$` |
| `unknown variable: xy` | 多字母下标未加引号 | 改为 `$P_("xy")$` |
| `unknown variable: vv` | 同上 | 改为 `$F_("vv")$` |
| `unknown symbol` | 符号名称错误 | 查阅 Typst 文档确认正确符号名 |
| `Error compiling HTML` | 整体语法错误 | 检查上述所有问题 |

## 4. 快速检查清单

在提交代码前，请检查以下内容：

- [ ] 单位符号（如 `rad`）是否用引号包裹
- [ ] 多字母下标是否用引号包裹
- [ ] 矩阵省略号是否指定方向（`dots.h`/`dots.v`）
- [ ] 表格中是否避免了直接使用数学符号
- [ ] 特殊符号（如 `times.circle`、`plus.minus`）是否使用正确名称
